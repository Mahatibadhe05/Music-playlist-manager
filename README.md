# Music Playlist Manager

A simple command-line music playlist manager built with Python. The application allows users to manage their personal playlists using JSON for persistent data storage.

## Features

- View playlist
- Add new songs
- Delete songs
- Persistent data storage using JSON
- Multi-user playlist support
- Automatic playlist creation for new users
- Simple command-line interface

## Technologies Used

- Python
- JSON
- File Handling
- OS Module

## Project Structure

```text
Music-playlist-manager/

├── common_playlist.json

```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Mahatibadhe05/music-playlist-manager.git
```

2. Navigate to the project folder

```bash
cd music-playlist-manager
```

3. Run the application

```bash
python multi_user_playlist.py
```
